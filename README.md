# CodeTogetherAI 🚀

CodeTogetherAI is a **real-time collaborative coding platform** that allows multiple users to join the same room and code together simultaneously. It is designed for pair programming, team collaboration, and remote coding sessions with smooth real-time synchronization.

---

## ✨ Features

- 🔥 Real-time collaborative code editing
- 👥 Multiple users can join the same coding room
- ⚡ Instant code synchronization using WebSockets
- 🔗 Room creation and joining via unique Room ID / link
- 🧠 Clean and simple UI for better focus
- 📱 Responsive design for different screen sizes
- 🛠 Built with a scalable architecture for further upgrades

---

## 🛠 Tech Stack

### Frontend
- TypeScript
- Vite
- TailwindCSS

### Backend
- Node.js
- WebSockets (real-time communication)

---

## 🚀 Installation & Setup

### 1) Clone the repository

```bash
git clone https://github.com/O-Aryan/CodeTogetherAI.git
cd CodeTogetherAI
````

### 2) Install frontend dependencies

```bash
npm install
```

### 3) Start the backend server

```bash
cd server
npm install
npm run dev
```

### 4) Start the frontend

Open a new terminal:

```bash
cd ..
npm run dev
```

---

## 🧪 Usage

1. Run both frontend and backend servers
2. Open the application in your browser
3. Create a new room or join an existing room
4. Share the room link/ID with others
5. Start coding together in real-time
